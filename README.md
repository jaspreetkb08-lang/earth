# SPACEEDU – 3D Earth Website 🌍

A modern and immersive **3D Earth landing page** built using **Three.js**, **GSAP**, HTML, CSS, and JavaScript.

This project features a rotating Earth model, animated cloud layers, dynamic star particles, and smooth scroll-based transitions for an engaging educational space-themed experience.

---

# 🚀 Features

- 🌎 Interactive 3D Earth using Three.js
- ☁️ Animated cloud layer with realistic atmosphere effect
- ✨ Dynamic starfield particle background
- 🎬 GSAP-powered scroll animations
- 📱 Fully responsive design
- 🧊 Glassmorphism navigation bar
- ⚡ Smooth cinematic transitions between sections
- 🎨 Modern typography using Google Fonts

# 🌍 How It Works

## 1. Three.js Scene Setup

The project initializes:

- Scene
- Camera
- Renderer
- Lighting System

inside `main.js`.

---

## 2. Earth Rendering

Earth textures include:

- Diffuse texture
- Normal map
- Specular map
- Cloud texture overlay

These textures are mapped onto sphere geometries to create a realistic Earth model.

---

## 3. Scroll Animations

Using GSAP + ScrollTrigger:

- Text fades in smoothly
- Earth rotates while scrolling
- Camera perspective changes dynamically
- Sections animate progressively

---

## 4. Responsive Layout

Media queries in `style.css` ensure:

- Mobile responsiveness
- Proper typography scaling
- Earth repositioning on smaller devices
